# NLP Emotion Detection System

## 📌 Overview
This project detects emotions from text using Machine Learning (SVM + TF-IDF).

## 🚀 Features
- Emotion classification from sentences
- Trained SVM model
- TF-IDF vectorization
- Label mapping for emotions

## 🧠 Tech Stack
Python, Scikit-learn, NLP

## 📊 Example
Input: "I am very happy today"  
Output: Happy 😊

## ⚙️ How to Run
pip install -r requirements.txt  
python app.py
